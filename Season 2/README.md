# GPA2 Program
*The GPA2 program is a simple Python program that calculates the Grade Point Average (GPA) of a student's academic records. The program prompts the user to input a series of scores, one at a time, and calculates the GPA based on the entered scores.*

## Getting Started
*To run the program, simply execute the 'gpa2.py' file using a Python interpreter. The program will prompt the user to enter scores, one at a time. To stop entering scores, type "exit" when prompted.*
