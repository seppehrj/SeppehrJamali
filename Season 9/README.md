# Bot_Telegram Program
**This bot has various features, such as game, age detection, voice, Max, ArgMax, QR code and photo, the menu is beautifully designed and very practical**
## Getting Started
To start the bot, you need to search for the bot @MrSeppehr_Bot in Telegram and start it.
## Prerequisites
- Python 3.x
## Installing


### [telebot]((https://github.com/eternnoir/pyTelegramBotAPI)) Library:
---
**You can create a bot with various features from this library**
To run the program, you need to install the telebot library, to install the command in the terminal:
```
pip install telebot
```
### [datetime] Library:
---
**This library is used to get the current date and time**
To run the program, you need to install the datetime library, to install the command in the terminal:
```
pip install datetime
```
### [gtts]((https://github.com/pndurette/gTTS)) Library:
---
**gTTS (Google Text-to-Speech), a Python library and CLI tool to interface with Google Translate's text-to-speech API. Write spoken mp3 data to a file, a file-like object (bytestring) for further audio manipulation, or stdout.**
To run the program, you need to install the gtts library, to install the command in the terminal:
```
pip install gtts
```
### [qrcode]((https://github.com/lincolnloop/python-qrcode)) Library:
---
***A standard install uses pypng to generate PNG files and can also render QR codes directly to the console.***
To run the program, you need to install the qrcode library, to install the command in the terminal:
```
pip install qrcode
```
